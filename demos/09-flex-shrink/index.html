flex-shrink

  • Accepts a numeric value
  • Decreases the width of the flex-items relative to the rest.
  • All flex-items in this example have flex-grow set to 1 and the flex-basis is set to 500px.
  • The highlighted item has flex-shrink set to 0, you can increase the flex-shrink value and the item decreases realtive to the rest.
