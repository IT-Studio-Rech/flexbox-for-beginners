flex-grow

  • Accepts a numeric value
  • Defines how much a item will grow relative to the rest of the items
  • By default flex-grow affects the width of the item (flex-direction: row) to affect the height of the item you need to change the flex-direction to the cross-axis
  • flex-items have by default the flex-grow property set to 0, so in this example every number higher than 1 will be treated as 1
